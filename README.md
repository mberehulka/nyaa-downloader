# Nyaa downloader

# Usage

- Login into your [AniList](https://anilist.co/) account in the window that shows up after starting the APP

- Add some anime to your watching list

- Select the anime (move with arrows, WASD keys or mouse scroll) and click Space

- Or use the A and D key change the watched episode

<img src="./assets/screenshots/1.png" width="100%"/>

- The programm will search in [Nyaa](https://nyaa.si/) RSS server for a torrent in 1080p of the next episode

- Select one of the results and click hit Space

<img src="./assets/screenshots/2.png" width="100%"/>

- The .torrent file will open in your default torrent APP

<img src="./assets/screenshots/3.png" width="100%"/>

- After that the episode will be marked as watched in your watching list

# Running

```
cargo run --release
```
